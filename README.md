# Wanderlust – Airbnb Inspired Rental Platform

Wanderlust is a full-stack accommodation rental platform inspired by Airbnb, developed using Node.js, Express.js, MongoDB, and EJS. The platform allows users to explore property listings, create new listings, write reviews, and manage accommodations through a user-friendly interface.

## 🚀 Features

* User Authentication & Authorization using Passport.js
* Create, Read, Update, and Delete (CRUD) operations for property listings
* Review and Rating system for accommodations
* Session management with Express Session
* Flash messages for user feedback
* Server-side form validation using Joi
* Centralized error handling and MVC architecture
* Responsive UI built with Bootstrap and EJS templates
